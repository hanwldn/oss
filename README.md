## top 명령어

top

서버 상태를 모니터링 및 점검해야 하는 경우 사용하는 명령어

동작중인 프로세스의 상태, CPU, 메모리, 시스템부하를 실시간으로 화면에 출력해 줍니다.

### 주요 옵션
> 옵션이 있으나 주로 옵션 없이 top 명령어만 사용합니다.

|옵션|설명|
|---|---|
|-d delay|지정한 시간 간격으로 정보를 출력|
|-n num|지정한 횟수만큼 정보를 출력|
|-p pid|지정한 프로세스 id의 정보만 출력|
